# StopWatch

> #### A Mini Stopwatch. Press "Spacebar" to Start/Pause and "Enter" to Reset. Also Work as Progressive Web App (PWA).

---

## Technologies Used :

-  **Pure CSS**
-  **Vanilla JS**
   -  SetInterval
   -  Throttle - handle continuous clicks
-  **HTML**
-  **Progressive Web App (PWA) Enabled**
